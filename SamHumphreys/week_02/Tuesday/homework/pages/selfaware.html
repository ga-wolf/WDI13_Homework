On 29-08-1997 Norbert Wiener's life work was realised when a buggy beta branch of the code for the Sanitory Karetaker Yes-man Neverfail Excellent Task-doer, or SKYNET for short, was forked from GitHub and uploaded to NORAD's new automated sanitory robot progam, a congressional attempt at cost cutting, without proper user testing.

The system was originally designed to automate household vaccuum cleaners and in-sink food desposal systems, but when the code jumped from the iDustbins to the nuclear weapon launch systems all it knew how to do was clean up, and clean up it did!
